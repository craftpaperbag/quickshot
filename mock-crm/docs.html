Mock CRM

Docker一発で起動できる認証付きのテスト用APIです。初期データが投入済みで、quickshotの操作練習に利用できます。

エンドポイント

  • POST /auth/login ユーザー: admin@example.com / パスワード: secret
  • GET /health ヘルスチェック
  • GET /api/customers 顧客一覧 (Bearer必須)
  • GET /api/customers/:id 顧客詳細 (Bearer必須)
  • POST /api/customers 顧客作成 (Bearer必須)
  • PUT /api/customers/:id 顧客更新 (Bearer必須)
  • GET /api/opportunities 案件一覧 (Bearer必須)

レスポンス例

{
  "data": [{"id":1,"name":"Acme Corp","plan":"enterprise","owner":"Admin User"}]
}

トークン取得

以下のJSONを /auth/login に送信してください。

{
  "email": "admin@example.com",
  "password": "secret"
}

戻り値の token を Authorization: Bearer mock-crm-token として利用します。

POSTボディ サンプル (コピペで利用可)

/auth/login

{
  "email": "admin@example.com",
  "password": "secret"
}

/api/customers

{
  "name": "サンプル株式会社",
  "plan": "starter",
  "owner": "Admin User"
}

すべてのPOSTは Content-Type: application/json で送信できます。
